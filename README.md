# AlignTask

A full-stack freelance marketplace platform where clients can post jobs, freelancers can submit proposals, and projects can be managed through a complete hiring workflow.

## Live Website

🔗 https://align-task-ruby.vercel.app

## Purpose

AlignTask was built to connect clients and freelancers through a modern freelancing platform. Each logged in user have their own dashboard depending on their roles. Clients can post tasks, review proposals, hire freelancers, and manage projects, while freelancers can browse jobs, submit proposals, and track their earnings. And Admin can track and manage all users, tasks and earnings.

---

## Features

### Authentication

* Email & Password Authentication
* Google Sign-In
* Protected Routes
* Role-Based Access Control

### Client Features

* Create Tasks
* Manage Posted Jobs
* View Freelancer Proposals
* Accept or Reject Proposals
* Stripe Payment Integration
* Track Project Status

### Freelancer Features

* Browse Available Tasks
* Search Tasks
* Filter by Category
* Submit Proposals
* View Proposal History
* Track Earnings
* Edit Profile

### Admin Features

* User Management
* Block / Unblock Users
* Monitor Platform Activity
